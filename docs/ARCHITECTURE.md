# HookForge Architecture

HookForge uses a Hook Kernel plus module architecture. The kernel is the Uniswap v4 hook entrypoint. Modules provide isolated adaptive behavior, and every module is bounded by caps, gas budgets, pause controls, and execution ordering.

## Layers

1. Uniswap v4 Hook Layer
2. Pool Intelligence Engine
3. Adaptive Logic Modules
4. AI Coordination Layer
5. Gamification and Evolution Layer
6. Developer Studio
